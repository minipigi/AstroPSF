# AstroPSF
AstroPSF는 소행성의 겉보기 등급 산출에 최적화된, PSF(Point Spread Function) 기반 천체 측광 프로그램입니다.

[프로그램 다운로드 홈페이지](https://sites.google.com/js.hs.kr/astropsv-psf/%ED%99%88)

# 주요 기능
- 측광 대상 및 비교성 자동 선택
- 측광 대상 및 비교성 수동 선택
- 버튼 클릭 한 번으로 측광 대상 겉보기 등급 산출
